# Thai Fund Engine v8.2

Thai Fund Engine คือเครื่องมือวิเคราะห์การลงทุนเชิงปริมาณ (Quantitative Investing)
ที่ทำงานบนเว็บเบราว์เซอร์ ออกแบบมาเพื่อทดสอบกลยุทธ์การจัดพอร์ต
การปรับพอร์ต (Rebalancing) และการค้นหาพารามิเตอร์ที่เหมาะสมผ่าน Batch Grid Search

## Key Features
- Batch Backtesting & Grid Search (Lookback, Rebalance)
- Momentum / Dual Momentum / Risk-based Strategies
- Cost-adjusted Performance (Transaction Cost & Expense Ratio)
- Risk Metrics: CAGR, Sharpe Ratio, Max Drawdown, Turnover
- Rolling Horizon Analysis (Lump Sum vs DCA)
- Client-side only (No Backend Required)
